1rem = 16px == 1em

In responsive web design, it's common to use a combination of 1em and 1rem, depending on the specific requirements of the layout and the desired behavior. 1rem is often used for setting the overall layout and structure of the page, while 1em is used for scaling individual elements within that structure.

html { font-size: 16px }

/* Responsive media */

body {font-size:1rem} 

@media (min-width: 600px) { body { font-size:1rem } /* 16px */ }
@media (min-width: 960px) { body { font-size:1.125rem } /* 18px */ }
@media (min-width: 1200px) { body { font-size:1.25rem } /* 20px */ }

/* scaling individual elements */

h1 { font-size: 2em }
h2 { font-size: 1.5em }
h3 { font-size: 1.25em }
h4 { font-size: 1em; /* = body */ }
h5 { font-size: .8em }
h6 { font-size: .7em }
