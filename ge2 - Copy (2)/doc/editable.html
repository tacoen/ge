Editable

The document.designMode property is a feature in JavaScript that allows you to enable or disable the ability to edit the entire HTML document content. When you set document.designMode to "on", the entire document becomes editable, similar to how a rich text editor or a WYSIWYG (What You See Is What You Get) editor would work. Just add:

<script>document.designMode = "on";</script>
